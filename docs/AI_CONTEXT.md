# AI Context: Portfolio Professional Architect

## Stack Técnico
- **Framework**: Next.js 15 (App Router)
- **Biblioteca**: React 19
- **Gestor de Paquetes**: Bun
- **Estilos**: Tailwind CSS
- **Animaciones**: Framer Motion (subtles, transiciones de 0.3s-0.5s)
- **Lenguaje**: TypeScript (Strict Mode)

## Reglas de Oro de Código
- **Idiomas**:
    - **UI y Contenido**: Español (estricto).
    - **Comentarios**: Español (para lectura rápida).
    - **Código (variables/funciones/archivos)**: Inglés (estándar).
- **Atómico y Modular**: Máximo 150 líneas por archivo. Extraer lógica a hooks o subcomponentes.
- **Server-First**: Usar React Server Components (RSC) por defecto. Solo usar `'use client'` para interactividad.
- **Clean Code**: Nombres descriptivos, tipos explícitos (prohibido el `any`).

## Estructura del Portfolio
- **Hero**: Bienvenida impactante con el logo moderno.
- **Proyectos**: Diseño Bento Grid.
    - `Ginialym`: ERP/POS SaaS.
    - `arca-sdk`: SDK Open Source para AFIP.
- **Logros**: Medalla Pull Shark de GitHub.
- **Links**: LinkedIn, GitHub, perfil de APX.

## Estética
- **Modo**: Dark Mode predominante.
- **Paleta**: Carbón profundo (#0A0A0A), acentos Zinc, degradados sutiles Electric Blue/Violet.
- **Tipografía**: Sans-serif refinada (Inter o Geist).
